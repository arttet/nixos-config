# 🛠️ Operations

This section covers the day-to-day management of your NixOS workstation. Unlike traditional operating systems where you might manually tweak files in `/etc`, here everything is managed through your Git repository.

## Key Tasks

### 🔄 [Updates & Rebuilds](./rebuild)

Learn how to apply changes from your local repository to the live system and how to perform manual system updates.

### 🧹 [Maintenance & Cleanup](./cleanup)

Nix keeps old versions of software by design. Learn how to manage these "generations" and reclaim disk space when needed.

### 💾 [Backups](./backups)

A guide on what data matters most and how to safely back up your personal files, secrets, and local machine state.

### 🚑 [Recovery & Rollbacks](./recovery)

If something breaks, NixOS makes it trivial to roll back. This page covers both instant rollbacks and deep recovery from a live ISO.
