# Runtime

Runtime docs describe how to build, start, stop, inspect, and test local NixOS
targets.

Installation prepares the host environment. Runtime commands operate on built
systems.

## Active Runtime

Milestone 002 keeps one active runtime target:

| Target | Purpose | State |
| --- | --- | --- |
| `guest` | Minimal reproducible VM for local testing and development. | Disposable |

Continue with [Guest VM](guest-vm.md).
